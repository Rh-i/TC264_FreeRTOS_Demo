# 串口协议标准（16字节）

## 1. 帧格式

| 字段     | 长度（字节） | 说明                        |
| -------- | ------------ | --------------------------- |
| 帧头     | 2            | `0xAA 0x55`                 |
| 命令     | 1            | 命令码                      |
| 数据长度 | 1            | 有效数据字节数（0～9）      |
| 数据     | 9            | 有效数据，不足补 `0x00`     |
| 校验     | 1            | 累加和低8位（覆盖前13字节） |
| 帧尾     | 2            | `0xBB 0x66`                 |

**总长度** = 16 字节（固定）

---

## 2. 校验算法

校验字节 = 前 13 字节累加和的低 8 位。

```
校验范围：帧头(2) + 命令(1) + 数据长度(1) + 数据(9) = 索引 0 ~ 12
```

帧尾（索引 14~15）不参与校验。

**Python 示例：**
```python
def calculate_checksum(frame_bytes):
    return sum(frame_bytes[0:13]) & 0xFF
```

## 3. 命令码定义

| 命令码 | 方向      | 功能                                   |
| ------ | --------- | -------------------------------------- |
| `0x01` | 主机→从机 | 设置目标速度（cm/s）                   |
| `0x02` | 主机→从机 | 查询当前速度                           |
| `0x03` | 主机→从机 | 速度模式：以指定速度运行指定时间（ms） |
| `0x04` | 主机→从机 | 紧急停止                               |
| `0x05` | 主机→从机 | 设置舵机角度（°）                      |
| `0x06` | 主机→从机 | 查询当前舵机角度                       |
| `0x07` | 主机→从机 | 给出key_a信号量，最外小灯              |
| `0x08` | 主机→从机 | 给出key_b信号量，第二小灯              |
| `0x09` | 主机→从机 | 给出key_c信号量，只响                  |

**响应机制：**
- 查询命令（`0x02`）响应：返回查询结果
- 设置/运动命令（`0x01`、`0x03`、`0x04`）响应：数据长度 = 1，数据[0] = 状态码（`0x00` 成功）

## 4. 数据字段格式（小端字节序）

| 命令                      | 数据长度 | 数据内容                                 |
| ------------------------- | -------- | ---------------------------------------- |
| `0x01` 设置速度           | 4        | `int32` 速度（cm/s）                     |
| `0x02` 查询速度（请求）   | 0        | 无                                       |
| `0x02` 查询速度（响应）   | 4        | `int32` 当前速度（cm/s）                 |
| `0x03` 速度-时间模式      | 8        | `int32` 速度（cm/s），`int32` 时间（ms） |
| `0x04` 停止运动           | 0        | 无                                       |
| `0x05` 设置舵机角度       | 4        | `int32` 角度值（°），范围 -35~+35        |
| `0x06` 查询舵机角度(请求) | 0        | 无                                       |
| `0x06` 查询舵机角度(响应) | 4        | `int32` 当前角度（°）                    |
| `0x07` key_a信号量        | 0        | 无                                       |
| `0x08` key_b信号量        | 0        | 无                                       |
| `0x09` key_c信号量        | 0        | 无                                       |

## 6. 约束条件

| 约束项   | 说明                                        |
| -------- | ------------------------------------------- |
| 帧长度   | 固定 16 字节                                |
| 帧头     | 必须为 `0xAA 0x55`，任一错误则丢弃整帧      |
| 帧尾     | 必须为 `0xBB 0x66`，任一错误则丢弃整帧      |
| 校验     | 前 13 字节累加和低 8 位，不匹配则丢弃整帧   |
| 数据填充 | 数据长度 < 9 时，剩余字节补 `0x00`          |
| 时间参数 | 1 ~ 2^31-1 毫秒（`uint32` 正值）            |
| 速度     | 有符号整型（`int32`），负值表示反方向       |
| 字节序   | 所有多字节数据均为**小端模式（LSB First）** |
