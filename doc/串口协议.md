# 串口协议标准（16字节，相对位置）

## 1. 帧格式

| 字段     | 长度（字节） | 说明                        |
| -------- | ------------ | --------------------------- |
| 帧头     | 2            | `0xAA 0x55`                 |
| 命令     | 1            | 命令码                      |
| 数据长度 | 1            | 有效数据字节数（0～9）      |
| 数据     | 9            | 有效数据，不足补 `0x00`     |
| 校验     | 1            | 累加和低8位（覆盖前13字节） |
| 帧尾     | 2            | `0xBB 0x66`                 |

**总长度** = 16 字节（固定）

---

## 2. 校验算法

校验字节 = 前 13 字节累加和的低 8 位。

```
校验范围：帧头(2) + 命令(1) + 数据长度(1) + 数据(9) = 索引 0 ~ 12
```

帧尾（索引 14~15）不参与校验。

**Python 示例：**
```python
def calculate_checksum(frame_bytes):
    return sum(frame_bytes[0:13]) & 0xFF
```

---

## 3. 命令码定义

| 命令码 | 方向      | 功能                                          |
| ------ | --------- | --------------------------------------------- |
| `0x01` | 主机→从机 | 设置目标速度（cm/s）                          |
| `0x02` | 主机→从机 | 查询当前速度                                  |
| `0x03` | 主机→从机 | 设置相对位置（相对于当前位置的偏移量，cm）    |
| `0x04` | 主机→从机 | 查询设定的相对位置的情况（查询当前位置）      |
| `0x05` | 主机→从机 | 速度模式：以指定速度运行指定时间（ms）        |
| `0x06` | 主机→从机 | 距离?时间模式：在指定时间内移动指定距离（cm） |
| `0x07` | 主机→从机 | 紧急停止                                      |
| `0x08` | 从机→主机 | 运动完成事件（主动上报）                      |

**响应机制：**
- 查询命令（`0x02`、`0x04`）响应：和设置的方式相同的返回，但是包头是和查询一致的
- 设置/运动命令（`0x01`、`0x03`、`0x05`、`0x06`、`0x07`）响应：数据长度 = 1，数据[0] = 状态码（`0x00` 成功）。
- 主动事件（`0x08`）：数据长度 = 0，无数据

---

## 4. 数据字段格式（小端字节序）

| 命令                     | 数据长度 | 数据内容                                 |
| ------------------------ | -------- | ---------------------------------------- |
| `0x01` 设置速度          | 4        | `int32` 速度（cm/s）                     |
| `0x02` 查询速度（请求）  | 0        | 无                                       |
| `0x02` 查询速度（响应）  | 4        | `int32` 当前速度（cm/s）                 |
| `0x03` 设置相对位置      | 4        | `int32` 偏移量（cm，正值向前，负值向后） |
| `0x04` 查询位置（请求）  | 0        | 无                                       |
| `0x04` 查询位置（响应）  | 4        | `int32` 当前位置（绝对坐标，cm）         |
| `0x05` 速度?时间模式运动 | 8        | `int32` 速度（cm/s），`int32` 时间（ms） |
| `0x06` 距离?时间模式运动 | 8        | `int32` 距离（cm），`int32` 时间（ms）   |
| `0x07` 停止运动          | 0        | 无                                       |
| `0x08` 运动完成事件      | 0        | 无                                       |

## 5. 完整通信示例（含正确校验字节）

> 符号说明：
> - 帧头 `AA 55` 固定
> - 帧尾 `BB 66` 固定
> - `XX` 表示校验字节（根据前 13 字节计算）

### 5.1 设置相对位置：向前移动 50 cm

**命令：** `0x03`  
**数据长度：** 4  
**数据：** `32 00 00 00`（小端 int32 = 50）

帧数据（索引 0~12）：
```
AA 55 03 04 32 00 00 00 00 00 00 00 00
```

**校验计算：**
```
0xAA + 0x55 + 0x03 + 0x04 + 0x32 + 0x00 + 0x00 + 0x00 + 0x00 + 0x00 + 0x00 + 0x00 + 0x00
= 0xAA + 0x55 + 0x03 + 0x04 + 0x32
= 0xAA + 0x55 = 0xFF
  0xFF + 0x03 = 0x102 → 低8位 0x02
  0x02 + 0x04 = 0x06
  0x06 + 0x32 = 0x38
```

校验字节 = `0x38`

**完整帧：**
```
AA 55 03 04 32 00 00 00 00 00 00 00 00 38 BB 66
```

### 5.2 设置相对位置：向后移动 30 cm

**数据：** `-30` 的小端 int32 = `E2 FF FF FF`

帧数据（索引 0~12）：
```
AA 55 03 04 E2 FF FF FF 00 00 00 00 00
```

**校验计算：**
```
0xAA + 0x55 + 0x03 + 0x04 + 0xE2 + 0xFF + 0xFF + 0xFF + 0x00 + 0x00 + 0x00 + 0x00 + 0x00
= 0xAA + 0x55 = 0xFF
  0xFF + 0x03 = 0x102 → 0x02
  0x02 + 0x04 = 0x06
  0x06 + 0xE2 = 0xE8
  0xE8 + 0xFF = 0x1E7 → 0xE7
  0xE7 + 0xFF = 0x1E6 → 0xE6
  0xE6 + 0xFF = 0x1E5 → 0xE5
```

校验字节 = `0xE5`

**完整帧：**
```
AA 55 03 04 E2 FF FF FF 00 00 00 00 00 E5 BB 66
```

### 5.4 距离?时间模式：向前移动 100 cm，用时 2000 ms

**命令：** `0x06`  
**数据长度：** 8  
**数据：**
- 距离 `100` → `64 00 00 00`
- 时间 `2000` → `D0 07 00 00`

帧数据（索引 0~12）：
```
AA 55 06 08 64 00 00 00 D0 07 00 00 00
```

**校验计算：**
```
0xAA + 0x55 + 0x06 + 0x08 + 0x64 + 0x00 + 0x00 + 0x00 + 0xD0 + 0x07 + 0x00 + 0x00 + 0x00
= 0xAA + 0x55 = 0xFF
  0xFF + 0x06 = 0x105 → 0x05
  0x05 + 0x08 = 0x0D
  0x0D + 0x64 = 0x71
  0x71 + 0xD0 = 0x141 → 0x41
  0x41 + 0x07 = 0x48
```

校验字节 = `0x48`

**完整帧：**
```
AA 55 06 08 64 00 00 00 D0 07 00 00 00 48 BB 66
```

## 6. 约束条件

| 约束项         | 说明                                                                                   |
| -------------- | -------------------------------------------------------------------------------------- |
| 帧长度         | 固定 16 字节                                                                           |
| 帧头           | 必须为 `0xAA 0x55`，任一错误则丢弃整帧                                                 |
| 帧尾           | 必须为 `0xBB 0x66`，任一错误则丢弃整帧                                                 |
| 校验           | 前 13 字节累加和低 8 位，不匹配则丢弃整帧                                              |
| 数据填充       | 数据长度 < 9 时，剩余字节补 `0x00`                                                     |
| 时间参数       | 1 ~ 2???1 毫秒（`int32` 正值）                                                         |
| 速度/距离/位置 | 均为有符号整型（`int32`），负值表示反方向                                              |
| 命令 `0x03`    | 偏移量立即生效，从机基于当前实际位置移动，速度使用当前目标速度（未设置则使用默认速度） |
| 字节序         | 所有多字节数据均为**小端模式（LSB First）**                                            |
